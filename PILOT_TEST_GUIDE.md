## 🎯 사전 준비 체크리스트

### 1. 데이터베이스 준비

#### ✅ 루브릭 업데이트 적용
```
1. Supabase Dashboard 접속
   URL: https://supabase.com/dashboard/project/yhcaliipxhzaqqvpmjjr

2. SQL Editor 메뉴 클릭

3. 새 쿼리 생성 (New query)

4. database/update_rubrics.sql 파일 내용 복사하여 붙여넣기

5. "RUN" 버튼 클릭

6. 결과 확인:
   - "=== 루브릭 업데이트 완료 ==="
   - "총 루브릭 항목: 8 개"
   - "범주별 항목 수: 내용: 3, 조직: 3, 표현: 2"
```

**주의**: 기존 채점 데이터(scores)가 모두 삭제됩니다!

#### ✅ 샘플 데이터 확인
```sql
-- 교사 확인
SELECT id, name, email, diagnosis_stage FROM teachers;

-- 에세이 확인
SELECT id, title, is_anchor, is_calibration FROM essays WHERE is_active = true;

-- 루브릭 확인
SELECT category, name, description FROM rubrics WHERE is_active = true ORDER BY display_order;
```

## 📝 테스트 시나리오

### 시나리오 1: 교사 온보딩 (신규 교사)

**목표**: 새로운 교사가 시스템에 처음 접속하여 6편 채점 완료

#### 단계
1. **로그인**
   - 교사 모드로 로그인
   - 대시보드 확인

2. **첫 채점**
   - "에세이 채점 시작" 버튼 클릭
   - 2단 분할 UI 확인
   - 왼쪽: 에세이 본문
   - 오른쪽: 8개 루브릭 (아코디언)

3. **루브릭별 채점**
   ```
   각 루브릭에서:
   - 아코디언 클릭하여 펼치기
   - 1점/2점/3점 기준 읽기
   - 점수 버튼 클릭
   - 자동으로 다음 항목 펼침
   ```

4. **채점 완료**
   - 8개 루브릭 모두 채점 (프로그레스 바 100%)
   - "제출" 버튼 클릭
   - 다음 에세이 자동 로드

5. **6편 채점 후**
   - 대시보드에서 통계 확인
   - "예비 진단" 레벨 달성
   - SE (표준 오차) 확인

#### 측정 항목
- [ ] 에세이 1편당 평균 채점 시간: ___분
- [ ] 루브릭 기준의 명확성 (1-5점): ___점
- [ ] UI 사용 편의성 (1-5점): ___점
- [ ] 아코디언 UI 만족도 (1-5점): ___점

---

### 시나리오 2: 채점 기준 학습

**목표**: 새로운 루브릭의 1점/2점/3점 기준 이해도 검증

#### 단계
1. **루브릭 기준 읽기**
   ```
   각 범주별로:
   - 내용 (주장, 이유, 근거)
   - 조직 (통일성, 응집성, 완결성)
   - 표현 (어휘·문장, 어문 규범)
   ```

2. **경계 사례 이해**
   - 1점 ↔ 2점 경계 기준
   - 2점 ↔ 3점 경계 기준

3. **실제 적용**
   - 동일한 에세이를 2명의 교사가 채점
   - 점수 일치율 확인

#### 측정 항목
- [ ] 루브릭 기준 이해도 (1-5점): ___점
- [ ] 1점/2점 경계 명확성 (1-5점): ___점
- [ ] 2점/3점 경계 명확성 (1-5점): ___점
- [ ] 교사 간 일치도: ___%

---

### 시나리오 3: 관리자 모드 (에세이 관리)

**목표**: 관리자가 에세이를 추가하고 앵커 설정

#### 단계
1. **로그인**
   - 관리자 모드로 로그인
   - 관리자 대시보드 확인

2. **에세이 추가**
   - "에세이 관리" 클릭
   - "새 에세이 추가" 버튼
   - 제목, 본문, 메타데이터 입력
   - 저장

3. **앵커 설정**
   - "앵커 관리" 메뉴
   - 에세이 선택
   - "앵커로 설정" 체크
   - 앵커 해설 작성 (1↔2, 2↔3 경계 근거)

4. **Blueprint 상태 확인**
   - 앵커 에세이: __/16편
   - 캘리브레이션: __/32편
   - 일반 에세이: __/100편

#### 측정 항목
- [ ] 에세이 추가 편의성 (1-5점): ___점
- [ ] 앵커 설정 명확성 (1-5점): ___점
- [ ] 관리자 UI 만족도 (1-5점): ___점

### 시나리오 4: MFRM 분석 실행

**목표**: 충분한 데이터 수집 후 MFRM 분석 실행

#### 사전 조건
- 최소 3명 교사 × 9편 에세이 × 8개 루브릭 = 216개 관측치

#### 단계
1. **관리자 대시보드**
   - "MFRM 분석" 메뉴 클릭

2. **분석 실행**
   - "새 분석 실행" 버튼
   - 분석 대기 (약 30초-2분)

3. **결과 확인**
   - 교사별 엄격성 (Severity)
   - 교사별 일관성 (Infit/Outfit)
   - 헤일로 효과
   - 범주 불균형

4. **교사 리포트 확인**
   - 교사 모드로 전환
   - "내 리포트" 메뉴
   - 메트릭 카드 4개 확인

#### 측정 항목
- [ ] 분석 수렴 여부 (converged): ___
- [ ] 분석 소요 시간: ___초
- [ ] 결과 표시 정확성: ___
- [ ] 리포트 이해도 (1-5점): ___점
