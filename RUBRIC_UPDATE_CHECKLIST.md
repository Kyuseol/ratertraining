## 🐛 문제 발생 시

### 루브릭 업데이트 실패
```
증상: SQL 실행 오류

해결:
1. 오류 메시지 확인
2. database/schema.sql에서 rubrics 테이블 구조 확인
3. 필요시 테이블 재생성
```

### UI에서 루브릭이 안 보임
```
증상: 채점 페이지에서 루브릭 0개 표시

해결:
1. 브라우저 새로고침 (Ctrl+F5)
2. 개발자 도구(F12) → Network 탭에서 API 호출 확인
3. Supabase에서 rubrics 테이블 확인
   SELECT COUNT(*) FROM rubrics WHERE is_active = true;
```

### 채점 제출 실패
```
증상: "제출" 버튼 클릭 시 오류

해결:
1. 개발자 도구(F12) → Console에서 오류 확인
2. Network 탭에서 API 응답 확인
3. 8개 루브릭 모두 선택했는지 확인
```
