## 💡 배운 점

### 기술적 도전
1. **R PATH 설정**: PowerShell에서 `$` 문자 이스케이프 (`pr`$run`)
2. **UUID 형식**: 8-4-4-4-12 형식 준수 필요
3. **Supabase RLS**: admins 테이블에 anon 역할 SELECT 권한 필요
4. **styled-jsx 제거**: Create React App에서 미지원, 일반 CSS로 변경

### 해결 방법
1. **실행 스크립트 생성**: .ps1, .bat 파일로 간편화
2. **UUID 재설계**: 올바른 형식으로 모든 샘플 데이터 수정
3. **RLS 정책 추가**: CREATE POLICY로 접근 권한 부여
4. **CSS 분리**: EssayForm.css 등 별도 파일로 분리
